# coc-yard

Port [vscode-yard](https://github.com/pavlitsky/vscode-yard) to coc.nvim

## Install

Installation by vim-plug and yarn

```vim
Plug 'weirongxu/coc-yard', {'do': 'yarn install --frozen-lockfile'}
```

<!-- `:CocInstall coc-yard` -->

## Commands

`:CocCommand extension.generateYard`
